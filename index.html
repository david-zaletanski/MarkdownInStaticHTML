# Using Showdown To use the showdown markdown to HTML converter you must do the following: 1. Create a Converter object. * Optionally provide Converter any Showdown extensions. 2. Call converter.makeHtml(MARKDOWN) where MARKDOWN is the markdown syntax text to convert to HTML. 3. Display the resulting HTML. Javascript using a `backquote` ( ` ): ``` function onConvertMarkup() { var converter = new showdown.Converter(); var markup = document.getElementById("markupTxtBox").value; var html = converter.makeHtml(markup); document.getElementById("htmlContent").innerHTML = html; return false; } ``` Javascript using indentation: function onConvertMarkup() { var converter = new showdown.Converter(); var markup = document.getElementById("markupTxtBox").value; var html = converter.makeHtml(markup); document.getElementById("htmlContent").innerHTML = html; return false; }
